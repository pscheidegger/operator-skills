# operator-skills

Reusable agent skills for action-oriented execution, knowledge management, research workflows, and human-agent collaboration.

## Skills

- `document-ingestion-skill`: normalize files and parser output into Markdown
- `social-media-ingestion-skill`: acquire and transcribe accessible media sources
- `knowledge-extraction-skill`: classify durable knowledge outcomes
- `integration-candidate-skill`: create reviewable proposals for existing targets
- `substance-guard`: quality-gate drafted content
- `action-first-skill`: format operational responses

See `skills/skill-composition.md` for routing, decision ownership, and composition order.
