# Action First Skill

A reusable agent skill that prioritizes actionable progress over explanatory verbosity.

## Core Idea

Action before explanation.

The goal is not to maximize information.
The goal is to maximize useful progress.

## Before

Long explanations, tangents, and multiple competing options.

## After

- Current status
- Clear steps
- A supported estimate when useful
- One next action when work remains

## Suitable For

- Claude Code
- Codex
- Cursor
- Local agents
- Project execution workflows

See `SKILL.md` for the complete behavior definition.
