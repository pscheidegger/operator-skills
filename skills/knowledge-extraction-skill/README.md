# Knowledge Extraction Skill

A reusable skill for transforming transient information into durable knowledge.

## Core Idea

Do not archive conversations.

Extract durable knowledge.

## Workflow

Input
↓
Knowledge Extraction
↓
Classification
↓
Decision
↓
Action

## Decisions

Each extraction should end with one of:

- discard
- create-reference
- create-knowledge
- create-task
- create-candidate
- update-existing

## Integration

This skill is designed to work together with the Integration Candidate Skill.
