# SubstanceGuard

A reusable quality-gate skill for making drafted content more specific, grounded, concise, and operationally useful.

## Core Idea

Optimize for signal, not polish.

SubstanceGuard reviews or rewrites content after the underlying source, knowledge, and governance decisions have been made. It does not replace ingestion, knowledge extraction, integration-candidate governance, or action-first response formatting.

## Typical Uses

- Review AI-generated documentation or plans
- Gate README, issue, or pull-request text
- Tighten research summaries and knowledge notes
- Mark unsupported certainty and fake completeness

See `SKILL.md` for the behavior definition and `examples/` for representative cases.
